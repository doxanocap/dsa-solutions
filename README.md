# DSA Solutions

![Go](https://img.shields.io/badge/go-%2300ADD8.svg?style=for-the-badge&logo=go&logoColor=white)
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

Репозиторий содержит коллекцию алгоритмических решений, исследований и материалов для подготовки к техническим интервью (Google, TikTok).

## 📂 Структура Проекта

### 🔹 [Google OA](./google-oa)
Разбор и решения задач с онлайн-ассессментов Google.
- **Темы:** Жадные алгоритмы, Динамическое программирование, Комбинаторика.
- **Проекты:** `greedy-decreasing-subsequences`, `digit-sum-ways`, `token-points`.

### 🔹 [TikTok OA](./tiktok)
Задачи и инструменты анализа для TikTok.
- **Темы:** Математическая оптимизация, Анализ метрик, Security Risk Analysis.

### 🔹 [LeetCode](./leetcode)
Избранные задачи с LeetCode.
- **Ключевые топики:** Графы (Dijkstra, MST), Интервалы, Битовые операции, Кучи (Heap).

## 🚀 Запуск

Для запуска тестов всего проекта:

```bash
go test ./...
```

Для конкретной директории:

```bash
cd google-oa
go test -v ./...
```

## 🛠 Технологии

- **Go**: 1.23+
- **Python**: Скрипты для анализа данных.
