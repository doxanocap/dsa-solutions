# Google OA Practice

В этом разделе собраны решения задач, похожих на те, что встречаются в онлайн-ассессментах Google (Online Assessment).

## Структура

- **[greedy-decreasing-subsequences](./greedy-decreasing-subsequences/solution.go)**
  Задача на разбиение массива. Необходимо разделить массив на минимальное количество строго убывающих подпоследовательностей, используя жадный подход (каждый раз выбираем максимально возможный следующий элемент).

- **[digit-sum-ways](./digit-sum-ways/solution.go)**
  Задача на комбинаторику/DP. Подсчет количества вариантов (например, билетов из 4 цифр), где сумма цифр равна заданному числу `S`. Решение использует динамическое программирование для оптимизации подсчета.

- **[token-points](./token-points/solution.go)**
  Задача на подсчет очков. Необходимо вычислить итоговый счет на основе последовательности токенов, учитывая бонусные правила (например, дополнительные очки за подряд идущие токены определенного типа).

## Запуск

Каждая подпапка содержит файл `solution.go` и соответствующий тест `solution_test.go`.
Для запуска тестов используйте:

```bash
go test ./google-oa/...
```
