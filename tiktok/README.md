# TikTok OA Solutions

Решения алгоритмических задач и инструментов для анализа, связанных с TikTok Online Assessment.

## Содержимое

- **[card_packets.go](./card_packets.go)**
  Алгоритм оптимизации пакетов карт. Использует математические методы (GCD, решето Эратосфена) для нахождения оптимального распределения карт в пакеты, минимизируя остаток.

- **[hash_divisors.go](./hash_divisors.go)**
  Решение, связанное с работой с делителями чисел и хешированием.

- **[security_risk.go](./security_risk.go)**
  Анализ уязвимостей или рисков безопасности на основе входных метрик.

- **[metric_analysis.py](./metric_analysis.py)**
  Python-скрипт для анализа метрик.


## Запуск

Go-файлы принадлежат пакету `tiktok`. Для запуска тестов используйте:

```bash
go test ./tiktok/...
```
